# React Counter Assignment

This is a simple React application built with Vite. It demonstrates React state management using the `useState` Hook.

## Features
- Increment the counter by 1
- Increment the counter after a 2-second delay
- Demonstrate React batching with "Increment Twice"
- Correctly increment by 2 using updater functions

## Concepts Covered
- useState Hook
- State as a snapshot
- Batched updates
- Asynchronous state updates with setTimeout

## How to Run
1. Open the project folder
2. Run `npm install`
3. Run `npm run dev`
